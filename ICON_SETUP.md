# Icon Setup

Create a folder called `images/` in your extension directory and add these icons:

## Option 1: Use Emoji as Icon
1. Create `images/` folder
2. Download emoji icons from: https://emojipedia.org/clipboard/
3. Save as `icon-16.png`, `icon-48.png`, `icon-128.png`

## Option 3: Generate Icons Online
Use: https://www.favicon-generator.org/
Upload an image or emoji and generate multiple sizes.

The extension will work without icons (shows default Chrome icon), but adding them makes it look professional.
